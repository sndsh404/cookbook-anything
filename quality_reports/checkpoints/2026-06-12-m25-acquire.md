# Checkpoint: M2.5 autonomous acquisition, license-gated

Date: 2026-06-12 (session 1)

## The numbers

Against a local seeded site (50 public pages + robots-forbidden /private/
tree + offsite links): 0 robots violations and 0 off-allowlist fetches,
verified from the SERVER side of the wire (privateHits=0); 50 pages archived
with origin_url + fetched_at front matter; rerun fetched 0 pages (HTTP
cache, robots.txt included). License gate: a Commons image verified via the
LIVE Wikimedia Commons API with auto-generated attribution; a page claiming
CC0 while the API proves CC BY-SA 4.0 is rejected; All Rights Reserved is
rejected and converted into a figure request; an asset record stripped of
its license is rejected by ca validate (both firewalls agree). A screenshot
ships inside a browser-chrome frame stamped with URL + access date.

## What shipped (runner/, TypeScript on node --experimental-strip-types)

- `acquire/robots.ts` robots.txt parser, longest-prefix allow/disallow.
- `acquire/infra.ts` HTTP cache, per-domain rate limit, budgets, audit
  trail (acquire_audit.jsonl), honest User-Agent.
- `acquire/html2md.ts` rung 1: static HTML to clean markdown + link
  extraction.
- `acquire/license.ts` the gate: tiers (PD/CC0 free; CC BY/BY-SA embed with
  non-removable attribution; NC/ND flag; unknown/ARR deny-and-redraw), and
  the claim-vs-API mismatch check.
- `acquire/acquire.ts` CLI: crawl (BFS, allowlist-scoped, archived) and
  asset (Commons metadata fetch, gate verdict, asset record or figure
  request).
- `testserver.mjs` the seeded crawl target with server-side hit counters.
- `figlib/screenshot_frame.py` F-14: chrome frame, URL + date stamp.

## Decisions

- Zero npm dependencies: Node 22 global fetch + --experimental-strip-types
  runs the TS directly. Playwright (ladder rung 2) deferred: nothing in the
  M2.5 done-criteria needs JS rendering, and the rung escalates only on
  failure anyway. Logged as a TODO for when a real target needs it.
- robots.txt responses are cached like any fetch so a rerun is truly
  zero-network.
